# 🌿 Vedika's Mehndi Arts

A modern and responsive website developed for **Vedika's Mehndi Arts** to showcase mehndi designs and services online.

## 📖 Overview

This project was created using HTML, CSS, and JavaScript with a focus on:

- Elegant and aesthetic design
- Mobile-friendly responsiveness
- Smooth user experience
- Interactive animations
- Professional presentation of mehndi artwork

The website serves as a digital portfolio, allowing visitors to explore different mehndi styles and learn more about the artist's work.

---

## ✨ Features

- Responsive design for mobile, tablet, and desktop
- Beautiful wedding-inspired UI
- Mehndi design gallery
- Smooth scrolling experience
- Scroll-based reveal animations
- Interactive buttons and sections
- Custom typography and styling
- Lightweight and fast-loading website

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Google Fonts
